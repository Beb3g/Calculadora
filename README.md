# Calculadora 🔢🔥🔥🔥
Uma calculadora 

Essa é minha primeira calculadora, bem simples, só o básico que tem as seguintes funções:  
  -1 (soma)  
  -2 (subtração)  
  -3 (multiplicação)  
  -4 (divisão)  
  -5 (potência)  
  -6 (raiz quadrada)            
  
  
  
  Para rodar ou abra o .exe ou use o .py direto com Python.  
  Use o .exe se não tiver Python instalado.
